# 贪吃蛇游戏逻辑

```
1、主逻辑更新速度每秒6次
   a.移动蛇
   b.移动蛇后吃食物
   c.吃到食物后长大
   d.检测游戏是否结束(碰撞)

2、食物逻辑每秒更新一次
   a.生成一个新的食物

3、主要事件及后续操作
   a.GameOverEvent 游戏结束：删除所有实体、创建一个小蛇
   b.GrowthEvent 蛇长大：增加一段蛇身
   c.蛇吃到食物：删除食物实体、发送GrowthEvent
   d.移动蛇：根据蛇头方向向前移动一格、检测碰撞并发送GameOverEvent
   e.键盘输入：调整蛇头方向
   
```
